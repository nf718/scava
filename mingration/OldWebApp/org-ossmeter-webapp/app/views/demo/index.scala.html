Projects

OSSMETER monitors the source code, issue trackers, and communication channels of open-source software (OSS). It's goal is to help you make decisions. If you're looking to adopt a particular OSS project, you can take a look at the metrics for the project to check whether your requirements will be met. E.g. if you care about having questions answered quickly, OSSMETER will tell you the average response time on the communication channels. If you're a member of an OSS project you can find out which aspects of your project are doing well and which could maybe do with a bit of improvement.

Take a look at the three projects below to get an idea of what kinds of insights you can gain.

ATL

ATL is a model transformation language and toolkit. In the field of Model-Driven Engineering, ATL provides ways to produce a set of target models from a set of source models.

View Project

EMF

The EMF project is a modeling framework and code generation facility for building tools and other applications based on a structured data model.

View Project

Graphiti

Graphiti is a graphics framework that enables rapid development of state-of-the-art diagram editors for domain models. Graphiti can use EMF-based domain models very easily but can deal with any Java-based objects on the domain side as well.

View Project

Compare Projects

When adopting OSS, there can be many projects that do similar jobs, and understanding which would be the best fit for you is a big challenge. To help with this selection process, we allow users to compare selected projects on the metrics that matter to them.

View Comparison Demo

User Account

OSSMETER will allow you to create a user account, providing you with new features on the website.

Personal Dashboard

Your dashboard is a place for you to keep track of the projects and metrics that matter the most to you.

Event Groups

Your dashboard allows you to create event groups. An event is simply a label and a date, and you group related events together. Event groups can then be overlaid on any plots on the OSSMETER website. This allows you to see whether a project reacted in some way to an event. For example, you can see whether the release of a new standard affected a project's development activity.
