Discover

OSSMETER monitors the source code, issue trackers, and communication channels of open-source software (OSS). It's goal is to help you make decisions. If you're looking to adopt a particular OSS project, you can take a look at the metrics for the project to check whether your requirements will be met. E.g. if you care about having questions answered quickly, OSSMETER will tell you the average response time on the communication channels. If you're a member of an OSS project you can find out which aspects of your project are doing well and which could maybe do with a bit of improvement.

Compare

When adopting OSS, there can be many projects that do similar jobs, and understanding which would be the best fit for you is a big challenge. To help with this selection process, we allow users to compare selected projects on the metrics that matter to them.

Adopt

After discovering and comparing the options, you are in a better position to adopt the project that best suits your personal or business needs.

Monitor

Users have a personal dashboard - a place for you to keep track of the projects and metrics that matter the most to you. You can set up alerts for specific metrics in order to be notified when that metric exceeds or falls below a given threshold. This can help you migrate away from a project if its quality begins to decline, or if you're a developer, help you find areas for improvement.

What is OSSMETER?

OSSMETER computes trustworthy quality indicators of open-source software by performing advanced analysis and integration of information from diverse sources including the project metadata, source code repositories, communication channels and bug tracking systems of OSS projects.

OSSMETER is not another OSS forge but instead it provides a meta-platform for analysing existing OSS projects that are developed in existing OSS forges and foundations such as SourceForge, Google Code, GitHub, Eclipse, Mozilla and Apache.
